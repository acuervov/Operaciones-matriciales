Este es un estudio computacional con el objetivo de comparar en rendimiento de diferentes librerias matriciales con el metodo de blocking a la hora de hacer dos operaciones matematicas entre matrices.

Hecho por Juan David Rodriguez y Andres Felipe Cuervo

Para hacer correr este programa se debe ejecutar el comando bash Codigo/run.sh, esto compilara todos los programas, los ejecutara y creara archivos txt con los resultados y luego sus respectivas graficas.

Si se quiere compilar y ejecutar un programa en especifico se puede hacer con el comando make Codigo/"operacion"_"metodo".txt

Se pueden ver los resultados del estudio en el archivo proyecto_intermedio_grupo_5.pdf
